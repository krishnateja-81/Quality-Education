What is C?

C is a general-purpose programming language created by Dennis Ritchie at Bell Laboratories in the early 1970s. It is considered one of the most influential programming languages and has had a profound impact on the field of computer science and software development.

Why Learn C?

  • It is one of the most popular programming languages in the world.
  • If you know C, you will have no problem learning other popular programming languages such as Java, Python, C++, C#, etc., as the syntax is similar.
  • C is very fast compared to other programming languages like Java and Python.
  • C is very versatile; it can be used in both applications and technologies.

Key Features of C

  • Procedural Language
  • Low-Level Language
  • Portability
  • Efficiency
  • Standard Library
  • Influence on Other Languages
  • Operating Systems
  • Popularity
  • Standardization

Input in C using scanf

In C programming, the scanf function is used for reading input from the user. It allows you to accept various types of data, such as integers, floating-point numbers, characters, and more. This tutorial will focus on using scanf to read an integer from the user.

scanf Function Syntax:


        #include <stdio.h>
    
        int main() {
            int num;
            printf("Enter an integer: ");
            scanf("%d", &num);
            printf("You entered: %d\n", num);
            return 0;
        }

Explanation:

The `scanf` function in C is used to read input from the user. It allows you to accept various types of data, such as integers, floating-point numbers, characters, and more. In this example, we'll focus on reading an integer.

  • Step 1: Include Header

    First, include the standard input/output library using #include <stdio.h>. This library provides functions like `printf` and `scanf` for input and output operations.

  • Step 2: Declare Variables

    Declare a variable named num of type `int`. This variable will store the integer entered by the user.

  • Step 3: Prompt for Input

    Use the `printf` function to display a prompt message, asking the user to enter an integer. The message "Enter an integer: " will be shown on the console.

  • Step 4: Read Input

    Utilize the `scanf` function to read an integer input from the user. The format specifier %d is used inside `scanf` to specify that an integer is expected. The &num part is used to store the entered integer in the num variable.

  • Step 5: Display Output

    Use the `printf` function again to display the entered integer. In the format string, %d is used as the format specifier for an integer. The value of num will be inserted into the output string.

  • Step 6: Return Statement

    The program ends with the return 0; statement, indicating successful execution.

C Comments

Understanding C Comments

In C, comments are used to provide explanatory notes within the source code. These comments are ignored by the compiler and are meant for human readers to better understand the code.

There are two common types of comments in C:

  • Single-Line Comments: Single-line comments begin with // and continue until the end of the line.
  • Multi-Line Comments: Multi-line comments are enclosed between /* and */ and can span multiple lines.

Single-Line Comments

In C, single-line comments start with // and extend until the end of the line.

// This is a single-line comment

They are used for short explanations or comments on a single line of code.

Multi-Line Comments

Multi-line comments, also known as block comments, start with /* and end with */.

/*
   This is a multi-line comment
   It can span multiple lines
*/

They are useful for longer explanations or comments that span multiple lines of code.

  Example:


          #include <stdio.h>
    
          int main() {
              // This is a single-line comment
              printf("Hello, World!\n"); // Print a message
    
              /*
                 This is a multi-line comment.
                 It can span across several lines.
                 Useful for providing detailed explanations.
              */
              return 0;
          }

Output:

Hello, World!

C Type Conversion

Definition and Information

Type conversion, also known as type casting, is the process of converting one data type into another in C. It's important for performing operations with mixed data types and ensuring that the data is appropriately represented.

C provides both implicit (automatic) and explicit (manual) type conversions to handle different scenarios.

Types of Type Conversion:

1. Implicit Type Conversion

Implicit type conversion, also known as type promotion, occurs automatically when a value of a lower-ranked data type is used in an expression with a higher-ranked data type.

2. Explicit Type Conversion

Explicit type conversion, also known as type casting, is performed manually by the programmer to change the data type of a value when needed. It's done using casting operators like (type).

Syntax of Type Conversion:


    // Implicit Type Conversion (Type Promotion)
    higher_data_type_variable = lower_data_type_value;
    
    // Explicit Type Conversion (Type Casting)
    new_data_type_variable = (new_data_type) old_data_type_variable;

Example of Type Conversion:


    #include <stdio.h>
    
    int main() {
        int integer_value = 42;
        double double_value;
    
        // Implicit Type Conversion (Type Promotion)
        double_value = integer_value;
    
        printf("Double Value: %lf\n", double_value);
    
        // Explicit Type Conversion (Type Casting)
        double_value = (double) integer_value;
    
        printf("Double Value (after casting): %lf\n", double_value);
    
        return 0;
    }

Output:

Double Value: 42.000000

Double Value (after casting): 42.000000

Definition and Information

In C, constants are values that do not change during the program's execution. They are used to represent fixed values like numbers, characters, or strings. Constants can be categorized into numeric constants, character constants, and string constants.

Constants provide a way to make code more readable and maintainable by using meaningful names for fixed values.

Character constants represent individual characters enclosed in single quotes.

Examples: 'A', '5', '%'

3. String Constants

String constants represent sequences of characters enclosed in double quotes.

Examples: "Hello, World!", "C Programming"

Assignment Operators

Assignment Operators in C

Assignment operators are used to assign values to variables.

Operator Description Example
= Assigns the value on the right to the variable on the left. a = 10;
+= Adds the value on the right to the variable on the left. a += 5; // Equivalent to a = a + 5;

Logical Operators

Logical Operators in C

Logical operators are used to perform logical operations such as AND, OR, and NOT.

Operator Description Example

|| (OR) Logical OR a || b
! (NOT) Logical NOT !x

Logical operators are often used in conditional statements to control program flow based on conditions.

Example of C Logical Operators:


            #include <stdio.h>

            int main() {
                int x = 5;

Bitwise Operators

Bitwise Operators in C

Bitwise operators are used to manipulate individual bits of data.

Operator Description Example
& (AND) Bitwise AND x & y
| (OR) Bitwise OR a | b
^ (XOR) Bitwise XOR x ^ y
~ (NOT) Bitwise NOT ~x
<< (Left Shift) Left Shift x << n
>> (Right Shift) Right Shift x >> n

Example of C Bitwise Operators:


            #include <stdio.h>

            int main() {
                int x = 5;   // Binary: 0101
                int y = 3;   // Binary: 0011

                int result;

                result = x & y; // Bitwise AND
                printf("x & y = %d\\n", result);  // Output: 1 (Binary: 0001)

                result = x | y; // Bitwise OR
                printf("x | y = %d\\n", result);  // Output: 7 (Binary: 0111)

                result = x ^ y; // Bitwise XOR
                printf("x ^ y = %d\\n", result);  // Output: 6 (Binary: 0110)

                result = ~x; // Bitwise NOT
                printf("~x = %d\\n", result);  // Output: -6 (Binary: 11111010)

                result = x << 2; // Left Shift by 2 bits
                printf("x << 2 = %d\\n", result);  // Output: 20 (Binary: 10100)

                result = x >> 1; // Right Shift by 1 bit
                printf("x >> 1 = %d\\n", result);  // Output: 2 (Binary: 0010)

                return 0;
            }

Output:

x & y = 1

x | y = 7

x ^ y = 6

~x = -6

x << 2 = 20

x >> 1 = 2

C Switch Statement

Understanding C Switch Statement

In C, the switch statement is used to make decisions based on the value of a variable or expression. It provides a way to select one of many code blocks to be executed.

The basic syntax of a switch statement is as follows:


            switch (expression) {
              case value1:
                // Code to execute if expression matches value1
                break;
              case value2:
                // Code to execute if expression matches value2
                break;
              default:
                // Code to execute if expression doesn't match any case
            }

The switch statement evaluates the expression and then checks it against each case. If a case matches, the corresponding code block is executed. If no cases match, the code in the default block (if present) is executed.

It's important to include break; statements after each case to exit the switch statement once a match is found.

Example of C Switch Statement:


      #include <stdio.h>

      int main() {
          int day = 3;

          switch (day) {
              case 1:
                  // Code for Monday
                  printf("Today is Monday\n");
                  break;
              case 2:
                  // Code for Tuesday
                  printf("Today is Tuesday\n");
                  break;
              case 3:
                  // Code for Wednesday
                  printf("Today is Wednesday\n");
                  break;
              default:
                  // Code for other days
                  printf("It's not a weekday\n");
          }

          return 0;
      }

Output:

Today is Wednesday

C While Loop

Understanding C While Loop

In C, the while loop is used for repetitive execution of a block of code as long as a specified condition is true. It's often used when you don't know in advance how many times the loop will run.

Here's the basic syntax of a while loop:


      while (condition) {
          // Code to be executed repeatedly
      }

The condition is a Boolean expression that determines whether the loop should continue or terminate. As long as the condition is true, the loop will keep executing.

Here's an example of a while loop:


      #include <stdio.h>

      int main() {
          int count = 1;

          while (count <= 5) {
              printf("Count: %d\n", count);
              count++;
          }

          return 0;
      }

What is a For Loop?

A "for" loop is a control flow statement in programming that allows you to execute a block of code repeatedly based on a specified condition. It consists of three parts:

  • Initialization: It is the initialization step where you initialize a counter variable.
  • Condition: The loop continues to execute as long as a specified condition is true.
  • Increment/Decrement: It is the step that increments or decrements the counter variable after each iteration.

Here's a basic example of a "for" loop in C:


        for (int i = 0; i < 5; i++) {
            // Code to be executed repeatedly
        }

Example:


        #include 

        int main() {
            for (int i = 1; i <= 5; i++) {
                printf("Iteration %d\n", i);
            }
            return 0;
        }

Benefits of Using For Loops

  • Efficient way to perform repetitive tasks.
  • Controlled and predictable execution.
  • Commonly used in iterating through arrays and collections.
  • Easy to read and understand.

What is a Break Statement?

The break statement is a control flow statement in programming that is used to exit a loop prematurely when a certain condition is met. It is commonly used in for loops, while loops, and switch statements.

Here's an example of how the break statement works:


        for (int i = 1; i <= 10; i++) {
        if (i == 5) {
        break; // Exit the loop when i is 5
        }
        printf("Iteration %d\n", i);
        }

What is a Continue Statement?

The continue statement is a control flow statement in programming that is used to skip the current iteration of a loop and proceed to the next iteration. It is commonly used in for loops and while loops.

Here's an example of how the continue statement works:


        for (int i = 1; i <= 5; i++) {
        if (i == 3) {
        continue; // Skip iteration when i is 3
        }
        printf("Iteration %d\n", i);
        }

            data_type array_name[array_size];

            int numbers[5]; // Declare an integer array of size 5

C Pointers

Understanding Pointers in C

In C, a pointer is a variable that stores the memory address of another variable. It allows you to work directly with memory locations and is a fundamental concept in C programming.

Pointers are declared by specifying the data type they point to, followed by an asterisk (*) and the pointer's name. Here's the syntax:


            data_type *pointer_name;

For example, to declare a pointer to an integer:


            int *ptr;

Pointers are used to:

  • Access and manipulate data indirectly.
  • Pass data efficiently to functions by reference.
  • Dynamically allocate memory for data structures like arrays and linked lists.

Pointer Examples:

Here are some examples to illustrate how pointers are used in C:


          #include <stdio.h>

          int main() {
              int num = 42;
              int *ptr = &num;

              printf("Value of num: %d\n", num);
              printf("Value pointed to by ptr: %d\n", *ptr);

              *ptr = 100;
              printf("Updated value of num: %d\n", num);

              return 0;
          }

Output:


          Value of num: 42
          Value pointed to by ptr: 42
          Updated value of num: 100

Pointer Arithmetic:

Pointers can be incremented or decremented to navigate through memory locations. This is useful when working with arrays or dynamic data structures.


          int numbers[] = {1, 2, 3, 4, 5};
          int *ptr = numbers;

          printf("First element: %d\n", *ptr);
          ptr++;

Default Parameters in C

Handling Default Parameters

C does not natively support default parameters in functions like some other programming languages. However, you can achieve similar behavior by using function overloading or by defining multiple versions of the function with different parameter lists.

Here's how you can simulate default parameters:


            #include <stdio.h>

            // Function with default parameters
            void printMessage(char message[], int count) {
                for (int i = 0; i < count; i++) {
                    printf("%s\n", message);
                }
            }

            int main() {
                // Call with one parameter
                printMessage("Hello" );//Error:too few arguments to function â€˜printMessageâ€™

                // Call with two parameters
                printMessage("Hi", 3);

                return 0;
            }

In this example, the function `printMessage` takes two parameters, `message` and `count`. However, if you want to provide a default `count`, you can create a separate function that only takes the `message` parameter:


            // Function with a default parameter
            void printMessageWithDefault(char message[]) {
                printMessage(message, 1);
            }

Now, you can call either `printMessage` with both parameters or `printMessageWithDefault` with just the `message` parameter.

Recursion in C

Understanding Recursion

Recursion is a programming technique where a function calls itself to solve a problem. In C, recursion is a powerful tool for solving complex problems by breaking them down into smaller, more manageable subproblems.

Key concepts of recursion:

  • Base Case: A condition that defines when the recursion should stop.
  • Recursive Case: The part of the function where it calls itself to solve a smaller subproblem.

Example of a recursive function to calculate factorial:


            #include <stdio.h>

            // Recursive function to calculate factorial
            int factorial(int n) {
                // Base case: factorial of 0 is 1
                if (n == 0) {
                    return 1;
                } else {
                    // Recursive case: factorial(n) = n * factorial(n-1)
                    return n * factorial(n - 1);
                }
            }

            int main() {
                int num = 5;
                int result = factorial(num);
                printf("Factorial of %d is %d\n", num, result);
                return 0;
            }

Output:

   Factorial of 5 is 120

In this example, the `factorial` function calls itself with a smaller value until it reaches the base case. Recursion allows us to express the factorial calculation naturally.
